# GyroMaze-STM32
이 프로젝트는 임베디드 실험의 텀 프로젝트 결과물입니다.

![gyromaze demo 002](resource/DEMO-002.gif)

### 개요

모바일 기기로 미로판을 조작하는 구슬 미로 게임입니다. 

#### 실험 장비

- STM32(Cortex-M3)
- JTAG(Joint Test Action Group)
- DSTREAM
- DS-5
- Android Device

#### 실험 소프트웨어

- Eclipse

### 사용 모듈

- bluetooth
- servo motor
- light sensor
- led
