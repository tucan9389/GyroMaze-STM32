# GyroMaze-STM32
이 프로젝트는 임베디드 실험의 텀 프로젝트 결과물입니다. 궁금하신 사항이 있으면 이슈에 남겨주세요! 질문과 제안은 언제나 환영입니다.

![gyromaze demo 002](resource/demo-001.gif)

## 개요

모바일 기기로 미로판을 조작하는 간단한 구슬 미로 게임입니다. 

## 실험 환경

### 실험 장비

- STM32(Cortex-M3)
- JTAG(Joint Test Action Group)
- DSTREAM
- Android Device

### 소프트웨어

- Eclipse for DS-5
- Android Studio
